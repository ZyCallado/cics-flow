# **App Name**: CICS Flow

## Core Features:

- Student Google OAuth: Implement secure Google OAuth for students, strictly restricting access to email addresses ending in @neu.edu.ph.
- Admin Credential Login: Provide a secure username/password login mechanism for administrators with robust credential verification.
- Role-Based Access Control: Define and enforce user roles (student, admin) to control access to various application features and data.
- Basic User Profile Display: Allow authenticated users to view their essential profile information upon successful login.
- Document Metadata Schema: Establish the fundamental database structure for document metadata including name, type, uploader, timestamp, and permissions.
- Audit Log for Access: Record all user login attempts and significant access events in a comprehensive activity log for security monitoring.
- AI Access Insight Tool: An AI tool for administrators to analyze the audit logs, identifying and highlighting unusual login patterns or potential security concerns.

## Style Guidelines:

- Light color scheme, professional yet inviting. Primary brand color: vibrant orange (#F2780D) for emphasis and call-to-actions, evoking dynamism and warmth.
- Background color: a very soft, warm off-white (#F5F1EF) providing a clean and inviting canvas for content.
- Accent color: a vibrant, rosy red (#EC4465) to highlight key information or interactive elements, creating contrast and visual interest.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, slightly tech-inspired, and impactful header text. Body font: 'Inter' (sans-serif) for clear, highly readable general text and navigation.
- Utilize simple, clean line-art icons that are clearly recognizable and professional, aligning with the educational and service-oriented nature of the portal.
- A responsive two-column layout optimized for both desktop and mobile, with clear separation between content and interactive elements. Use card-based UI components for distinct information blocks.
- Incorporate subtle hover effects on interactive elements like buttons and links, and gentle transitions for page or component changes, enhancing user feedback without being distracting.
